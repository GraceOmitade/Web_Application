CSS Properties

Below you'll find a list of all CSS properties from the Cascading Style Sheets 1 specifications up to Cascading Style Sheets 3 specifications. CSS properties are used to apply styles to structured documents, such as fonts, colors and spacing. Used correctly, CSS can be a powerful web design tool.

Note that not all browsers/user agents support all CSS properties, you should try to test your pages in as many browsers as possible.
